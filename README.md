🚀 Nexus – Investor & Entrepreneur Collaboration Platform
Nexus is an advanced frontend platform designed to bridge the gap between startups and investors. This project was enhanced as part of the Advanced Frontend Internship to include real-time collaboration tools, secure financial tracking, and an interactive user experience.

🔗 Project Links
Live Deployment: https://nexus-public-mbfv.vercel.app

GitHub Repository: https://github.com/InsaSaleem/nexus-public

✨ Advanced Features Implemented
During this 3-week internship, I successfully implemented the following milestones:

📅 Week 1: Scheduling & Setup
Integrated FullCalendar: Added a functional meeting scheduling system.

Availability Management: Users can now view and manage their professional slots.

Consistent UI: Established a responsive grid and a professional color palette.

📞 Week 2: Video Calling & Document Chamber
WebRTC Video Call UI: Integrated a dedicated video conferencing modal with audio/video toggles.

Document Processing: Built a 'Document Chamber' for contract management.

E-Signature Mockup: Added a signature pad for finalizing deals digitally.

💰 Week 3: Payments, Security & Final Polish
Digital Wallet: Implemented a mock payment UI with balance tracking and transaction history.

OTP Security: Added a multi-step 2FA login mockup for enhanced security.

Guided Platform Tour: Integrated React Joyride to provide first-time users with a step-by-step walkthrough of the dashboard.

Role-Based UI: Developed distinct views for Entrepreneurs and Investors to streamline their specific workflows.

🛠️ Tech Stack Used
Frontend: React.js, TypeScript, Tailwind CSS
Icons & UI: Lucide React, Radix UI
Libraries: React Joyride, FullCalendar, PeerJS

Deployment: Vercel

🚀 How to Run Locally
1. Clone the repository:
git clone https://github.com/InsaSaleem/nexus-public.git

2. Install dependencies:
npm install

3. Run the development server:
npm run dev
